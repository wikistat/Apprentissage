## [Apprentissage Machine / Statistique](\http://wikistat.fr)

L'objectif de ces tutoriels, présentés sous forme de calepins ([*jupyter notebooks*](http://jupyter.org/)), est d'introduire, à l'aide de scénarios codés en R ou Python, les méthodes et techniques de la statistique ou de la *Sciences des données*. Ces sénarios détaillent des exemples ou cas d'usage  d'analyses de données "réelles".  Ce dépôt est consacré à l'apprentissage machine / statistique. D'autres scénarios sont proposés sur [wikistat.fr](http://wikistat.fr/) ainsi que que dans les dépôts de ce site. Les méthodes sont exposées dans les vignettes de [wikistat](http://wikistat.fr/).

Chaque dossier concerne un jeu de données et contient un calepin (*notebooks*) au format .ipynb à télécharger et ouvrir dans *Jupyter*.
